Design Document
Project: Personal Portfolio Website
Owner: Akash
Version: 1.0

1. Overview
Purpose:
This document defines the visual direction, layout structure, interaction behavior, and UI guidelines 
for building a minimalistic, modern, and unique portfolio website inspired by contemporary portfolio designs.

Design Goal:
Create a clean, bold, high-contrast portfolio that highlights personality, projects, and technical expertise 
without unnecessary clutter.
2. Design Vision & Principles
- Minimal and spacious layout
- Strong typography hierarchy
- Generous whitespace
- Smooth and subtle animations
- High visual clarity
- Content-first design approach
3. Visual Style Guidelines
Color Palette:
- Background: Dark / near-black
- Primary Accent: Electric blue or soft purple
- Text: White / Light gray
- Borders/Shadows: Subtle neutral tones

Typography:
- Primary Font: Modern sans-serif (Inter / Poppins style)
- Hero Heading: Extra Bold, Large
- Section Titles: Bold
- Body Text: Regular with comfortable line spacing
- Labels: Light weight
4. Page Layout Structure
Hero Section:
- Full-screen introduction
- Name and professional tagline
- Short introduction (2 lines max)
- Primary CTA buttons (View Projects / Contact)

About Section:
- Short personal introduction
- Optional portrait
- Skills and focus areas

Skills Section:
- Grid layout of skill badges
- Minimal hover effects
- No progress bars

Projects Section:
- Card-based or grid layout
- Project image
- Title and description
- Tech stack tags
- GitHub / Live links

Contact Section:
- Simple contact form
- Social media links
- Minimal footer
5. Interaction & Motion Design
- Smooth scrolling
- Fade-in animations on scroll
- Subtle hover transitions
- Minimal micro-interactions
- No excessive animations
6. Responsive Design
- Mobile-first approach
- Vertical stacking on smaller screens
- Clean navigation (hamburger menu for mobile)
- Optimized spacing for readability
7. Accessibility & Best Practices
- High contrast ratios
- Semantic HTML structure
- Keyboard navigation support
- Image alt text
- Fast loading (optimized images and fonts)
